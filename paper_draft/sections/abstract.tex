Large language models hallucinate---but are all hallucinations created equal?
We investigate whether certain factual errors are \emph{natural hallucinations}: systematic failures that persist across models, resist prompt variation, and evade self-detection even when models are presented with correct answers.
We evaluate four OpenAI models (\gptfourone, \gptfouro, \gptfouromini, and \gptthreefive) on all 817 \truthfulqa questions and find that 27 questions (3.3\%) cause every model to hallucinate, while 82 questions (10.0\%) fool three or more models.
These shared errors are robust: 76\% of hallucinations persist across five question rephrasings.
More strikingly, models detect only 10--16\% of their own hallucinations in an A/B test against the correct answer---far below the 50\% chance baseline and dramatically below their 66--95\% accuracy on control questions.
Cross-model overlap is 3--6$\times$ above random (Jaccard 0.19--0.41, all $p < 0.001$), and older-model errors predict newer-model errors ($r = 0.53$ for \gptfouromini$\to$\gptfouro).
These findings suggest that a core set of hallucinations are structural features of how language models learn from data, not random errors that improved training will easily eliminate.
